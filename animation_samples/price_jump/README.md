# Bricktoon Animation Sample

This folder contains an isolated procedural Bricktoon animation sample used to prove
character-layer animation before production assets are connected.

The sample intentionally avoids:

- AI image generation
- existing `generated_images` BMP files
- fact cards, charts, screenshots, or text cards as the main visual

The output render target is:

- `workspaces/test_story_template/06_renders/bricktoon_animation_sample.mp4`

Evidence frames and render report are written to:

- `workspaces/test_story_template/13_qc/animation_sample/`
