# Generalized Asset Library

This folder is the reusable organized bricktoon asset/reference source for future stories.

Use it for:

- people
- workers
- office workers
- villains
- buildings
- offices
- rooms
- interiors
- hospitals
- prisons
- police
- police rooms
- houses
- cars
- trucks
- pets
- facial expressions

Workflow:

1. Store reusable organized images in the category folders.
2. Select what a topic needs in `04_assets/reference_manifest.json`.
3. Run the reference-sync stage.
4. Review the synced workspace references.
5. If required references do not exist yet, generate them with ComfyUI and add the approved versions back into this library for future reuse.
