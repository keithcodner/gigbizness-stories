# Generalized Reference Library

Use this folder for broad visual style references and thumbnail-quality inspiration images.

Purpose:

- store premium visual references that define the target bricktoon look
- keep reusable style references outside a single topic workspace
- sync selected images into `workspaces/<topic>/04_assets/reference_images/`

Recommended use:

1. Place style and composition references here.
2. Keep filenames descriptive.
3. Add the topic-specific picks to `04_assets/reference_manifest.json`.
4. Run the reference-sync stage to copy them into the workspace.

This folder is for style, composition, lighting, and look references.
It is not the main organized subject library.
