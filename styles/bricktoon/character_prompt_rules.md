# Bricktoon Character Prompt Rules

- Describe characters as original blocky toy cartoon figures.
- Use role-first costume cues like blazer, headset, clipboard, mover gloves, or generic badge.
- Keep outfits repeatable across scenes for continuity.
- Avoid official toy-brand language, exact minifigure terms, or branded accessories.
- Add front view, side view, and three-quarter view when building references.
